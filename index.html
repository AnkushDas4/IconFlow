Developer Manual

Everything you need to integrate IconFlow into your production apps. We support simple HTML embeds, React components, and CSS backgrounds.

html

HTML & No-Code

The easiest method. Works in Webflow, WordPress, Framer, and raw HTML.

<!-- Just use it as an image source -->
<img
  src="https://iconflow.vercel.app/api/render?q=google&color=000&size=24"
  alt="Google Logo"
  width="24"
/>
code

React & Next.js

Build a dynamic icon component that changes based on props.

export const Icon = ({ name, color }) => (
  <img
    src={`https://iconflow.vercel.app/api/render?q=${name}&color=${color}&size=24`}
    className="w-6 h-6"
  />
);

// Usage: <Icon name="github" color="black" />
css

CSS Backgrounds

Perfect for pseudo-elements (::before) or custom buttons.

.btn-icon {
  background-image: url('https://iconflow.vercel.app/api/render?q=menu&color=white&size=20');
  background-repeat: no-repeat;
  background-position: center;
}
